⚡ Pokémon Endpoints

GET
/api/v2/pokemon/{id or name}/
Full Pokémon data — stats, moves, sprites, abilities, types
›
Key Response Fields
idintegerNational Dex number
namestringPokémon identifier (lowercase)
base_experienceintegerXP yield when defeated
heightintegerHeight in decimetres
weightintegerWeight in hectograms
types[]arrayslot, type.name, type.url
stats[]arraybase_stat, effort, stat.name
abilities[]arrayability.name, is_hidden, slot
moves[]arraymove.name + version_group_details
spritesobjectfront_default, back_default, shiny, official-artwork
speciesNamedAPIResourceLink to species endpoint
JavaScript Example
// Fetch Pokémon by name or ID
async function getPokemon(nameOrId) {
  const res = await fetch(
    `https://pokeapi.co/api/v2/pokemon/${nameOrId}`
  );
  const p = await res.json();

  return {
    name:  p.name,
    types: p.types.map(t => t.type.name),
    stats: Object.fromEntries(
      p.stats.map(s => [s.stat.name, s.base_stat])
    ),
    sprite: p.sprites.front_default,
  };
}
💡 Use /api/v2/pokemon-species/{id} for flavor text, egg groups, gender rate, and the evolution chain URL. The /pokemon endpoint only provides battle data.
GET
/api/v2/pokemon/{id}/encounters
All wild encounter locations for a Pokémon
› 
// Get all locations where Pikachu can be caught
const res = await fetch(
  'https://pokeapi.co/api/v2/pokemon/pikachu/encounters'
);
const encounters = await res.json();
// Returns array of: { location_area, version_details[] }
// version_details has: version.name, encounter_details[].method
//   .min_level, .max_level, .chance
GET
/api/v2/ability/{id or name}/
Ability details, descriptions, and Pokémon that have it
› 
const ability = await fetch(
  'https://pokeapi.co/api/v2/ability/intimidate'
).then(r => r.json());

ability.effect_entries   // English effect text
ability.pokemon          // Pokémon with this ability
ability.generation.name  // "generation-iii"

💥 Move Endpoints

GET
/api/v2/move/{id or name}/
Move power, accuracy, PP, type, category, and effects
›
powerinteger|nullBase damage (null for status)
accuracyinteger|nullHit chance % (null = always hits)
ppintegerPower Points (uses)
priorityintegerSpeed priority (-6 to +5)
type.namestringElemental type
damage_class.namestringphysical / special / status
effect_entries[]arrayShort and full effect text
metaobjectailment, drain, healing, crit_rate, flinch_chance…
learned_by_pokemon[]arrayAll Pokémon that learn this move
const move = await fetch(
  'https://pokeapi.co/api/v2/move/earthquake'
).then(r => r.json());

move.power          // 100
move.accuracy       // 100
move.pp             // 10
move.type.name      // "ground"
move.damage_class.name // "physical"
move.target.name    // "all-other-pokemon"

🎒 Item Endpoints

GET
/api/v2/item/{id or name}/
Item description, cost, fling effects, held-by Pokémon
› 
const item = await fetch(
  'https://pokeapi.co/api/v2/item/choice-band'
).then(r => r.json());

item.cost                  // Buy price in Pokédollars
item.fling_power           // Power when used with Fling
item.held_by_pokemon[]     // Pokémon that hold it naturally
item.effect_entries[]      // Short effect + full description
item.flavor_text_entries[] // In-game descriptions per version
item.sprites.default       // Item bag sprite URL
item.category.name         // "held-items"
GET
/api/v2/item-category/{id or name}/
All items belonging to a bag category
› 
// Get all berries in the Medicine category
const cat = await fetch(
  'https://pokeapi.co/api/v2/item-category/medicine'
).then(r => r.json());

cat.items   // [{ name, url }, …]
cat.pocket  // Which bag pocket: "medicine"

🔥 Type Endpoints

GET
/api/v2/type/{id or name}/
Full damage relation matrix + all Pokémon of this type
› 
const type = await fetch(
  'https://pokeapi.co/api/v2/type/dragon'
).then(r => r.json());

// Damage relations (attack perspective)
type.damage_relations.double_damage_to   // [dragon]
type.damage_relations.half_damage_to     // [steel]
type.damage_relations.no_damage_to       // [fairy]

// Damage relations (defense perspective)
type.damage_relations.double_damage_from // [ice, dragon, fairy]
type.damage_relations.half_damage_from   // [fire, water, grass, electric]
type.damage_relations.no_damage_from     // []

type.pokemon  // All Pokémon with this type (huge array!)

🔄 Evolution Endpoints

GET
/api/v2/evolution-chain/{id}/
Full evolutionary line with all conditions
› 
// 1. Get species to find chain URL
const species = await fetch(
  'https://pokeapi.co/api/v2/pokemon-species/eevee'
).then(r => r.json());

const chainUrl = species.evolution_chain.url;

// 2. Fetch the chain
const chain = await fetch(chainUrl).then(r => r.json());

// chain.chain is the root node:
// { species, evolution_details, evolves_to[] }
// Recurse evolves_to[] to walk the tree

// evolution_details[0] fields:
// min_level, item, trigger, min_happiness,
// held_item, location, time_of_day, known_move…

🗺️ Location Endpoints

GET
/api/v2/location-area/{id or name}/
Pokémon encounters, encounter methods, and conditions for one area
› 
const area = await fetch(
  'https://pokeapi.co/api/v2/location-area/pallet-town-area'
).then(r => r.json());

area.pokemon_encounters.forEach(enc => {
  console.log(enc.pokemon.name);         // "pidgey"
  enc.version_details.forEach(vd => {
    console.log(vd.version.name);        // "red"
    vd.encounter_details.forEach(e => {
      console.log(e.method.name);        // "walk"
      console.log(e.chance);             // encounter rate %
      console.log(e.min_level, e.max_level);
    });
  });
});

🎮 Games Endpoints

const gen = await fetch(
  'https://pokeapi.co/api/v2/generation/1'
).then(r => r.json());

gen.pokemon_species.length // 151

gen.types                   // types introduced this gen
gen.version_groups          // Red/Blue/Yellow version groups

/api/v2/pokedex/{id or name}/

› 
const dex = await fetch(
  'https://pokeapi.co/api/v2/pokedex/kanto'

dex.pokemon_entries.forEach(e => {
  console.log(e.entry_number);       // Regional # (1–151)

});

🔧 Utility & Pagination

Resource Lists & Pagination

// All list endpoints support ?limit and ?offset
const res = await fetch(
  'https://pokeapi.co/api/v2/pokemon?limit=20&offset=40'
).then(r => r.json());

res.count    // Total available (e.g. 1302)
res.next     // URL to next page (or null)
res.previous // URL to prev page (or null)
res.results  // [{ name, url }, …]

Unnamed Resource Lists

Some endpoints (evolution-chain, contest-effect, machine) return resources with only a url (no name).

// These endpoints have no names in their list
const chains = await fetch(
  'https://pokeapi.co/api/v2/evolution-chain?limit=10'
).then(r => r.json());

chains.results
// [{ url: "…/evolution-chain/1/" }, …]
// (no "name" field)

Language Endpoint

// 36+ supported languages for localizing names
const langs = await fetch(
  'https://pokeapi.co/api/v2/language'
).then(r => r.json());
// Includes: en, ja, fr, de, es, it, ko, zh-Hans, zh-Hant, …

// Most resources have "names" arrays:
const name_ja = pokemon.species.names
  .find(n => n.language.name === 'ja')?.name;
