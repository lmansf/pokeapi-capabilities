💡 What Can You Build?
PokéAPI is free, RESTful, and requires no API key. Here are common applications.
📱
Pokédex App
Build a full Pokédex with stats, types, sprites, evolutions, and Pokédex entries for all 1,025 Pokémon.
/api/v2/pokemon/{name}
⚔️
Battle Damage Calculator
Calculate move damage with full type effectiveness, STAB, and base power data for competitive play.
/api/v2/move/{name}
🏆
Team Builder
Analyze team type coverage, find weaknesses, and suggest complementary Pokémon to round out a roster.
/api/v2/type/{name}
🗺️
Location Guide
Show players exactly where to catch a Pokémon, with encounter rates, conditions, and level ranges per game.
/api/v2/pokemon/{id}/encounters
🧬
Breeding Planner
Look up egg groups, compatible parents, and growth rates to plan efficient Pokémon breeding chains.
/api/v2/egg-group/{name}
🎮
Game Companion
Deliver generation-specific Pokédex data, version-exclusive Pokémon, and regional form information.
/api/v2/generation/{id}

🚀 Getting Started
No API key, no rate limit sign-up — just fetch and go.
💡 All responses are JSON. The base URL is https://pokeapi.co/api/v2. Resources can be looked up by ID or by name. 
// Fetch a Pokémon by name (or ID)
const response = await fetch('https://pokeapi.co/api/v2/pokemon/pikachu');
const pokemon = await response.json();

console.log(pokemon.name);           // "pikachu"
console.log(pokemon.base_experience); // 112
console.log(pokemon.types);           // [{ slot: 1, type: { name: "electric" } }]
console.log(pokemon.stats[0]);        // { base_stat: 35, stat: { name: "hp" } }
console.log(pokemon.sprites.front_default); // sprite URL

// Paginate resource lists
const list = await fetch('https://pokeapi.co/api/v2/pokemon?limit=20&offset=0');
const data = await list.json();
// data.count  → total Pokémon
// data.results → [{ name, url }, …]
